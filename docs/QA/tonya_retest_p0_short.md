# Ретест P0 для Тони (короткий прогон)

Цель: быстро проверить последние правки по качеству ответов, без перегруза таблицей.

## Как сдавать результаты (без таблицы)
- Отправляй в чат 1 сообщением на кейс.
- Формат: `RT-0X`, `✅/❌`, 1 фраза «что ожидалось / что вышло», 2-3 скрина подряд.
- Если ❌: добавь коротко «что именно не так» и уровень `S1/S2/S3`.

Шаблон:
`RT-0X | ✅/❌ | ожидалось: ... | факт: ... | S1/S2/S3`

## Что тестируем сейчас (7 кейсов)

1. `RT-01` Убрали хвост «Можно спросить»
- Шаги: новый диагноз по фото.
- Ожидаем: в основном ответе нет блока `Можно спросить:` и списка вопросов от бота.

2. `RT-02` Ответ стал короче и чище
- Шаги: новый диагноз + 1-2 уточнения в чате.
- Ожидаем: без «простыни», без повторов одинаковых абзацев/чеклистов.

3. `RT-03` Нет дубля чеклиста по воде/грунту
- Шаги: кейс, где бот и так пишет блок «Что уточнить перед выводами по воде/грунту».
- Ожидаем: этот блок не повторяется второй раз тем же текстом.

4. `RT-04` Память коротких ответов: статус влаги -> грунт -> запах
- Шаги:
  1) написать `слегка влажно`;
  2) написать `торф + перлит`;
  3) написать `запаха нет`.
- Ожидаем: бот подтверждает полученное и каждый раз просит только недостающее (не задаёт заново всё сразу).

5. `RT-05` Память работает в другом порядке
- Шаги:
  1) написать `торф + перлит`;
  2) написать `мокро`;
  3) написать `запаха нет`.
- Ожидаем: аналогично, только недостающие уточнения по шагам.

6. `RT-06` Fallback при нерелевантной реплике
- Шаги: после диагноза написать что-то вне контекста, например `привет`.
- Ожидаем: короткий fallback без «простыни» и с CTA `Спросить ассистента`.

7. `RT-07` Регрессия low-confidence
- Шаги: кейс с уверенностью ниже 65%.
- Ожидаем: бот просит минимум 2 уточняющих фото и не показывает «Запланировать обработку/Спросить про препараты» до досъёмки.

## Что важно отметить отдельно
- Если бот снова «уходит в гниль/фунгициды» без достаточных оснований.
- Если снова появляются повторы текста блоками.
- Если бот повторно спрашивает уже подтверждённые данные (влажность/грунт/запах).
