# Юр. комплаенс и согласия (РФ)

## Версии документов
- Политика ПДн: версия 1.0 (30.12.2025) — https://agronom.offonika.ru/privacy
- Публичная оферта: версия 1.0 (30.12.2025) — https://agronom.offonika.ru/offer

Версии задаются через env:
- `PRIVACY_VERSION`
- `OFFER_VERSION`
- `AUTOPAY_VERSION`
- `MARKETING_VERSION`

## Сценарии согласий в боте
- `/start`: единый экран согласий (ПДн + Оферта) с кнопкой «Принимаю и продолжаю».
- Первое фото: если нет согласий — показывается тот же экран; обработка не запускается.
- Перед оплатой: если нет согласий — экран согласий, затем оплата.
- Автопродление: отдельное подтверждение перед оплатой с автоплатежом.

## Хранение согласий
Согласия фиксируются в двух таблицах:
- `consent_events` — неизменяемый журнал действий (accept/revoke)
- `user_consents` — текущий статус согласия по пользователю

Минимальный набор полей:
- `doc_type`: `privacy` | `offer` | `autopay` | `marketing`
- `doc_version`
- `status` (для `user_consents`)
- `occurred_at` / `updated_at`

## Отзыв согласия
Отзыв для автопродления производится командой `/cancel_autopay`.
Для остальных типов — планируется отдельная команда/процесс (например, `/consent`).
